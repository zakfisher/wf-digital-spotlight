Testing 101 - A/B Split Best Practice

If you can, you should conduct A/B split tests. Split testing helps validate or disprove you assumptions in a controlled environment, ideally isolating variables, and allow for learnings to be applied to future marketing efforts. Below are some guidelines to make sure you have valid, clean tests.

  • Test 1 variable at a time. Simplified tests eases identification of variable of influence.

  • Make sure you're testing a tactic that can be repeated. For example, inclusion of brands or % offer vs $ offer.

  • Wait at least 24 hours before declaring a winner.

  • Make sure your results are statistically significant. Significance calculators are a great tool.

  • Repeat tests to increase confidence in results.
